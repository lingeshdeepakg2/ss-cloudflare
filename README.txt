== Description ==
 To provide a simple interface that allows us automatically create and return tokens and rules for a domain.

== Installation ==

 1. Install plugin by clicking Pluings > Add New
 2. upload the plugin by clicking Upload plugin button
 3. Select the pluign zip file to be install
 4. Activate the plugin by clicking activate pluign 
 
== Access Plugin Settings  ==

  1. Access the pluign settings page by going to Tools > SS Cloudflare

== Features ==

 - Provide the ability to verify the domain's validity, generate a token, and include options for incorporating WAF rules for the website.
